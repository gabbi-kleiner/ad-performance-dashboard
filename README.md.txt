# Digital Advertising Performance Dashboard

This project simulates a dashboard analyzing key metrics for digital advertising campaigns across multiple channels.

Metrics included:

- CPM (Cost per 1000 Impressions)
- CTR (Click Through Rate)
- ROAS (Return on Ad Spend)

## Tools Used

- HTML/CSS for dashboard structure
- Chart.js for interactive charts
- JavaScript for data representation

## Key Insights

- Retargeting campaigns generally show higher CTR and ROAS.
- Awareness campaigns generate higher impressions but lower direct conversions.
- Visualizing metrics helps optimize budget allocation.

## Future Improvements

- Pull data directly from APIs (e.g., TikTok Ads, Meta Ads)
